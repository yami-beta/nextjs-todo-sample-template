# nextjs-todo-sample-template

## コマンド

依存 package インストール

```bash
$ yarn install
```

開発サーバ起動

```bash
$ yarn dev
```

Lint, Format

```bash
$ yarn lint
$ yarn format
```

テスト

```bash
$ yarn test
```

型チェック

```bash
$ yarn type-check
```
